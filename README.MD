# Generator


To run code:

Windows:
```
py -3.11 -m venv venv
```
```
venv\Scripts\activate
```
```
pip install -r requirements.txt
```
```
python main.py
```
Linux/MacOS:
```
python3.11 -m venv venv
```
```
source venv/bin/activate
```
```
pip install -r requirements.txt
```
```
python main.py
```
